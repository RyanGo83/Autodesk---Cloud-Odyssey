# Autodesk: A Cloudy Odyssey

[View app: Autodesk: A Cloudy Odyssey](https://ai.studio/apps/drive/1CbFPtZJsPWBl6am11gFS5EMrwiRIZJn9)

A snarky interactive timeline exploring the perpetual rebranding of Autodesk's cloud platforms, featuring a countdown to the next inevitable name change.

## Run Locally

**Prerequisites:** Node.js

1. Install dependencies:
   `npm install`
2. Run the app:
   `npm run dev`
3. Open your browser at `http://localhost:3000`
